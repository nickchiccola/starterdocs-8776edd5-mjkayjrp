# Documentation Site

This is your documentation site powered by Hugo.

## Getting Started

1. Add your markdown files to the `content/` directory
2. Push changes to trigger automatic sync
3. Your docs will be searchable and indexed automatically

## Structure

- `content/` - Your markdown documentation files
- `config.toml` - Hugo configuration
- `archetypes/` - Content templates
